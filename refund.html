Return & Refund Policy

Last updated: 26 November 2025

1. Eligibility for Return

We accept returns only in the following cases:

  • Product received in damaged condition
  • Wrong item received (different model / color than ordered)
  • Product is not working on arrival (dead on arrival)

You must raise a return request within 7 days of delivery with clear photos / videos showing the issue.

2. Conditions for Return
  • Product should be in original condition as received.
  • All accessories, manuals, charging cables, and box (if available) should be returned.
  • Physical damage due to mishandling is not covered.
3. Refund Process

Once your returned product is received and inspected, we will notify you about the approval or rejection. If approved, your refund will be initiated to the original payment method (bank account / card / wallet).

Refund timelines:

  • UPI / Wallet – usually within 2–4 working days
  • Bank Cards / Netbanking – usually within 5–7 working days
4. Order Cancellation

Orders can be cancelled only before they are shipped. Once shipped, you can request a return after delivery based on the conditions mentioned above.

5. Non-Returnable Items

Certain items may not be eligible for return due to hygiene or manufacturer policies. Such information, if applicable, will be mentioned on the product page.

6. How to Raise a Return / Refund Request

To raise a return or refund request, please contact our support with your:

  • Order ID
  • Registered mobile number / email
  • Clear description of the issue
  • Photos / video of the product and packaging
